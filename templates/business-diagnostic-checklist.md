# Business Diagnostic Checklist

Use this checklist quarterly (or when lead flow feels "off") to find gaps in how your business handles leads and customers.

Score each item: **Yes** / **Partial** / **No** / **N/A**

---

## Lead capture and response

| # | Question | Score | Notes |
| --- | --- | --- | --- |
| 1 | Every inbound channel (form, email, phone, social) lands in one trackable place | | |
| 2 | First response SLA is defined (e.g. within 1 business day) | | |
| 3 | Auto-reply confirms receipt and sets expectations | | |
| 4 | Required intake fields capture budget, timeline, and problem summary | | |
| 5 | Spam and low-fit inquiries are filtered without ignoring legitimate leads | | |

---

## Qualification and prioritization

| # | Question | Score | Notes |
| --- | --- | --- | --- |
| 6 | Leads are scored or tagged using documented criteria | | |
| 7 | Hot leads get a different response path than nurture leads | | |
| 8 | Someone owns qualification — not "whoever is free" | | |
| 9 | Disqualified leads receive a polite, timely response | | |

---

## Follow-up and handoffs

| # | Question | Score | Notes |
| --- | --- | --- | --- |
| 10 | Follow-up cadence is documented (not ad hoc) | | |
| 11 | No lead sits untouched more than [X] days without a reason | | |
| 12 | Sales → delivery handoff includes written scope and contacts | | |
| 13 | Post-sale onboarding has a checklist | | |

## Data and privacy

| # | Question | Score | Notes |
| --- | --- | --- | --- |

## Scoring guide

Count **No** and **Partial** items:

| Result | Action |
| --- | --- |
| 0–3 gaps | Maintain; refine documentation |
| 4–7 gaps | Pick top 2 gaps; fix in next 30 days |
| 8+ gaps | Schedule working session; prioritize lead response and follow-up |
